# Recursion Review
## Another Example

```python
def print_chars(word, i):
    if i == len(word):
        return
    print(word[i])
    print_chars(word, i + 1)

print_chars("elephant", 3)
```

Output:
```python
# H
# e
# l
# l
# o
```

## What happens if a recursive function has no base case?

- ( ) It's no longer considered recursive
- (x) It will keep calling itself forever, resulting in an infinite loop
- ( ) Alan Turing comes back from the dead to haunt the author

## What is printed if I call print_chars('elephant', 3)?

- (x) phant
- ( ) elephant
- ( ) ele
- ( ) ant
